# Loan Approval Prediction using SHAP and LIME

A Machine Learning project that predicts loan approval status using a Random Forest Classifier and explains model predictions using SHAP and LIME Explainable AI techniques.

## Features
- Data Preprocessing
- Feature Encoding
- Random Forest Classification
- Model Evaluation
- SHAP Explainability
- LIME Explainability
- Feature Importance Analysis

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- SHAP
- LIME
- Matplotlib
- Google Colab

## Dataset
Loan Approval Prediction Dataset

## Results
The model predicts whether a loan application will be approved or rejected and provides interpretable explanations for each prediction using SHAP and LIME.
